# Как установить

Папку ru-RU поместить в папке с игрой по пути:
<ПУТЬ_К_ИГРАМ_STEAM>/common/Pathway/i18n/

В этой же папке есть другие переводы:
- de-DE
- en-US
- fr-FR
- ja-JP
- zh-CN

Теперь можно запускать игру и в настройках выбирать язык "Русский".
